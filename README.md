# Claude Environment - Sustenta Futuro

This package includes:
- `CLAUDE.md` root instructions for Claude Code

## Recommended next steps
1. Copy `CLAUDE.md` to the root of your repo.
2. Copy `.claude/agents/` into the repo.
3. Review `specs/month1-mvp-spec.md` and adjust the fields/statuses.
4. Start Claude Code in the repo and work from the spec first.
